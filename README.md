# Bunsvr

## Synopsis

Barebone http/https Bun server template repository.

## Installation

```
# Clone the repository
git clone https://github.com/DmmDGM/bunsvr my-project
cd my-project

# Install packages
bun i

# Starts server
bun .
```

## Configuration

```.env
# Server port.
PORT=3000
```
